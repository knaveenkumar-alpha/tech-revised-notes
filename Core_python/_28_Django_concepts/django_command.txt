Django project commands:
========================

> python --version 

> pip --version 

> pip install django / if you have requirements.txt file CMD pip install -r requirements.txt

> pip install requests

> mkdir <project_name>
 
> cd <project_name>

> django-admin startproject projectName

> python manage.py startapp projectApp

> python manage.py createsuperuser

if you have pipenv working fine go for pipenv or leave it

> python manage.py makemigrations   ===> if you change model.py file otherwise Not required 

> python manage.py migrate      ====> this command will create tables in DB

> python manage.py runserver    =====> this command will run our application

To stop our server we have to press  
> ctrl + c
